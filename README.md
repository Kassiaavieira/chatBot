# IA Chatbot com Hugging Face e OpenAI
Este projeto é uma integração de APIs de IA usando o NestJS para criar um chatbot simples. O objetivo é gerar respostas automáticas a partir de mensagens usando modelos de linguagem natural, como o EleutherAI/gpt-neo-1.3B da Hugging Face.

## 🛠️ Tecnologias Utilizadas
- **NestJS: Framework para desenvolvimento backend.**
- **Hugging Face: Para integração com modelos de IA como gpt-neo-1.3B.**

## 🚀 Como Rodar o Projeto
1- Clone este repositório:
```bash
git clone https://github.com/Kassiaavieira/chatBot.git
```
2- Instale as dependências:
```bash
npm install
```
3- Configurar a Chave de API
```bash
HUGGING_FACE_API_KEY=seu_token_huggingface
```
4- Rodar o Servidor
```bash
npm run start
```

## 🧠 Sobre o Projeto
Este é um projeto simples que estou usando para aprender a integrar APIs de IA com NestJS. Ele utiliza modelos de linguagem para gerar respostas automáticas e foi uma forma de explorar essas tecnologias.
